Introduction

This documentation provides an overview of web development basics.

We will cover essential topics including HTML, CSS, and JavaScript.

console.log('Hello, world!');

Follow along to understand the building blocks of web development.

HTML Basics

HTML (HyperText Markup Language) is the foundation of web pages.

It defines the structure and content of a web page.

  • Elements
  • Attributes
  • Tags
<div>Hello</div>
CSS Basics

CSS (Cascading Style Sheets) controls the visual presentation of a web page.

It allows you to apply styles such as colors, fonts, and layouts.

  • Selectors
  • Properties
  • Media Queries
body { color: black; }
JavaScript Basics

JavaScript adds interactivity to web pages.

It is a powerful scripting language used for web development.

  • Variables
  • Functions
  • Events
function greet() { alert('Hi!'); }
